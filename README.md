
# AI Resume Analyzer (Flask)

## 📌 Project Overview
This project is a simple web-based Resume Analyzer built using Flask. It analyzes resume text, detects technical skills, counts words, and generates a basic score.

## ⚙️ Features
- Resume text input
- Skill detection (Python, Flask, HTML, CSS, etc.)
- Word count analysis
- Simple scoring system

## 🛠️ Tech Stack
- Python
- Flask
- HTML
- CSS

## 🚀 How to Run
1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

## 🎯 Purpose
This project is built for learning Flask backend development and understanding how basic AI-like logic can be implemented in web applications.
